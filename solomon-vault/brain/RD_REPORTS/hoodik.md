# RD Report: Hoodik

**URL:** https://hoodik.io
**Type:** Self-hosted encrypted cloud storage
**Priority:** 🟢 Nice to have

---

## What it is

Hoodik is a self-hosted cloud storage platform with **end-to-end encryption** — files are encrypted in the browser before upload, the server never sees plaintext. Built with **Rust and Vue**. Designed for easy Docker installation.

**Key features:**
- End-to-end encryption (client-side, browser-based)
- Self-hosted (full control, own data)
- Docker install
- Public link sharing
- Optional 2FA
- Lightweight, fast uploads/downloads
- Low RAM usage

Posted by Camille Roux (Founder of HumandCoders) on April 17, 2026.

---

## What we'd use it for

Could be a **self-hosted Dropbox alternative** for Solomon OS infrastructure:
- Store client files securely
- Encrypted file sharing with staffing clients
- Backup for Solomon Vault

Fits the "privacy-first, self-hosted" ethos of Solomon OS.
