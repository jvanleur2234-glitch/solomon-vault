## 6 Free Go-To Resources for UI/Product Designers

| Resource | URL | Use For |
|---|---|---|
| Motion/Animation | https://60fps.design | Animations, micro-interactions |
| Patterns | https://catalogue.projectsbyif.com | Design patterns, UX patterns |
| AB Tests | https://abtest.design | Experiment ideas, conversion optimization |
| Inspiration | https://designspells.com | Visual inspiration, design direction |
| Breakdowns | https://uxsnaps.com | Real product UI teardowns |
| UI Components | https://uiplaybook.dev | Component library reference |

## When to Use

- **Designing any app/landing page** → Check 60fps.design for animation direction first
- **Building new features** → Check catalogue.projectsbyif.com for proven patterns
- **Improving conversion** → Check abtest.design for tested experiment ideas
- **Getting unblocked creatively** → Check designspells.com for inspiration
- **Teardown existing UIs** → Check uxsnaps.com for how real products handle specific problems
- **Component reference** → Check uiplaybook.dev for UI patterns and component examples

## Key Principle

Stop with the blue-purple gradient "Vibe Coding" aesthetic. These resources represent a higher standard for UI quality.
