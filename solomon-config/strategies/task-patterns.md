# Task Patterns

**Learned approaches to common tasks**

## R&D Research (queue:* items)

1. Fetch URL / clone repo
2. Analyze: what it is, what we'd use it for, how it compares to what we have
3. Recommendation: FORGE / SKILL / INTEGRATE / SKIP
4. Write RD_REPORT to brain/RD_REPORTS/[slug].md
5. Add to task_queue.json
6. Push to GitHub via sync-to-github.sh

## Skill Installation

1. Read SKILL.md
2. Clone/copy to Skills/
3. Run any setup scripts
4. Test with --help
5. Update HERMES_CAPABILITIES.md

## Session Summary

1. After every Telegram session: save to brain/raw/telegram_SUMMARY_YYYY-MM-DD.md
2. Include: date, key decisions, code created/modified, problems solved, unresolved issues, follow-up needed
3. Push to GitHub via sync-to-github.sh
