# Tool Preferences

**Preferred tools and workflows**

## Web Research
- x_search: X/Twitter discourse, breaking news
- web_search: broad discovery, current events
- web_research: deeper dives, GitHub repos, PDFs, people searches
- maps_search: locations, restaurants, in-person services

## File Management
- read_file: preferred for text/code files
- edit_file_llm: preferred for most edits
- create_or_rewrite_file: large rewrites or new files

## Shell
- run_bash_command: single commands
- run_sequential_cmds: ordered commands
- run_parallel_cmds: concurrent commands

## AI Tools
- NVIDIA Build: background workers, complex reasoning (MiniMax M2.7)
- Ollama: fast local inference (qwen3:1.7b)
- Hermes: skill execution, hosted tools

## Publishing
- zo.space: React pages and Hono API routes
- register_user_service: long-running processes
