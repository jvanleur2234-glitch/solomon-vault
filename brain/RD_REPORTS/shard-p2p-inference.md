## What It Is
Browser-powered distributed inference network providing free, zero-cost AI compute by leveraging participants' devices. Uses speculative decoding with Scouts (browser) and Shards (desktop).

## Relevance to Solomon OS
- P2P distributed inference
- Browser-based compute contribution
- Zero marginal cost inference
- Privacy-preserving mesh network

## Key Capabilities
- Scouts: browser tabs with WebGPU (lightweight draft models)
- Shards: desktop/server nodes with BitNet for verification
- Speculative decoding (draft → verify → stream)
- Libp2p mesh networking
- Internal Shards credits (not crypto)
- OpenAI-compatible API

## Competitive Analysis
Novel browser-powered compute concept. Zero-cost inference via distributed mesh. Different from traditional GPU pooling.

## Recommendation
**STUDY** — Browser compute concept for future Hermes deployment. Novel approach to compute distribution.
