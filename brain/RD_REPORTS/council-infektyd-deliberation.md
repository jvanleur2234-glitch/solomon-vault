# council — Multi-Agent Deliberation (infektyd/OpenClaw)

## What It Is
OpenClaw skill for parallel multi-agent deliberation across specialized AI personas (WORKHORSE, CREATIVE, etc.). Produces auditable transcripts, optional Discord visibility.

## Key Signals
- **License:** MIT
- **Lang:** Python 3.10+
- **Requires:** xAI API key

## Core Features
- Parallel deliberation across Grok/xAI personas
- Self-coordination without fixed runtime roles
- Auditable transcripts of deliberations
- Verdict mode (fastest single-round) + Deliberation mode (multi-round synthesis)
- Discord posting for human visibility
- Structured result envelope for calling agent
