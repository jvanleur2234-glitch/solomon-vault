## What It Is
A declarative, multi-agent AI framework with YAML-based workflow and agent definitions. Model-agnostic, supports P2P sharing, and includes a browser-based Studio IDE.

## Key Features
- Declarative YAML workflow and agent definitions
- Model-agnostic: OpenAI, Gemini, Anthropic, Ollama, LM Studio
- MCP (Model Context Protocol) for tool/external service connections
- P2P encrypted sharing of agents/engines — no central API server
- Agent Orcha Studio web dashboard with visual composer
- Built-in knowledge graph + vector store (SQLite-based)
- Browser sandbox with Chrome DevTools Protocol
