# RD Report: AI Counsel — True Multi-Agent Deliberation via MCP Server

**Forked to:** https://github.com/jvanleur2234-glitch/ai-counsel  
**License:** MIT  
**Stars:** active, deliberation-focused  
**Relevance:** multi-agent deliberation, consensus, adversarial verification  

## What It Is
AI Counsel is a true deliberative consensus MCP server where AI models (Claude, Codex, Gemini, Ollama) debate and refine positions across multiple rounds. Each model sees others' responses and changes its position — not just parallel aggregation.

## Key Features
- **Multi-round debate:** models engage in actual back-and-forth deliberation
- **Conference mode:** 3+ models with structured voting and confidence levels
- **Evidence-based:** models can read files, search code, run commands to ground decisions
- **Early convergence:** auto-stops when consensus is reached (saves API costs)
- **Full transcripts:** markdown exports with AI-generated summaries
- **Fault tolerant:** individual adapter failures don't halt deliberation
- **MCP server:** works with Claude Code, Codex, any MCP-compatible agent
