## What it is

Captain Claw is a full-featured AI agent platform with a terminal/web UI, multi-agent orchestration, autonomous cognitive architecture, and 44 built-in tools. Ships with Flight Deck — a management dashboard for agent teams.

## Key Capabilities

- **Flight Deck Dashboard:** Agent Forge (LLM designs specialized teams), Agent Council (structured deliberation), Fleet Communication, Director Panel
- **Agent Council:** Structured multi-agent deliberation — debates, brainstorms, reviews, planning. Moderated rounds with moderator synthesis + voting. MIT license.
- **Cognitive Architecture:** 5-layer memory (Working/Semantic/Deep/Insights/Nervous System), autonomous dreaming/tension-tracking/maturation, 7 cognitive modes
- **Multi-Model Support:** OpenAI, Claude, Gemini, Ollama, OpenRouter + ChatGPT OAuth
- **Orchestrator/DAG Mode:** Decompose tasks into dependency graphs, parallel execution
- **44 built-in tools**

## What we'd use it for

- Agent Council deliberation for business decisions — superior to our current council approaches
- Multi-agent team orchestration for complex Solomon OS tasks
- Fleet communication pattern for Hermes sub-agents

## Comparison to what we have

- More sophisticated deliberation (Agent Council) than our current council setups
- Full cognitive architecture with autonomous processes — deeper than Hermes's current memory layers
- Flight Deck provides UI we lack
