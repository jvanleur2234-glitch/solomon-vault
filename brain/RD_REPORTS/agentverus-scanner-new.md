# RD Report: agentverus/agentverus-scanner

## Summary
Open-source TypeScript security tool that analyzes AI agent skill files to detect trust and safety risks. Focuses on prompt injection, data exfiltration, and 10+ ASST threat categories.

## Relevance to Solomon OS
- **Skill file scanning**: Directly applicable to Hermes skills ecosystem
- **Trust scoring**: Could power Solomon OS skill trust framework
- **AGENTS.md protection**: Detects tampering with agent configuration files

## Key Features
- Prompt injection detection
- Data exfiltration detection
- 10+ ASST threat categories
- Capability contract checks
- Behavioral risk scoring
- Workspace config tampering detection (AGENTS.md, SOUL.md, CLAUDE.md)
- Structured trust reports with severity caps
