## What It Is
Minimalist CLI tool for AI agents to interact with browsers. Based on Playwright with a simplified API surface for autonomous agents.

## Key Features
- **Lightweight CLI:** Simple commands for browser automation
- **Accessibility tree snapshots:** Agents see structured page state
- **Element references:** Click/interact via stable refs
- **Screenshots:** Page capture
- **Cross-platform:** macOS, Linux, Windows

## Differentiation from vercel-labs/agent-browser
- Vercel version is Rust-based (faster, native)
- This version is Node.js based (easier to extend)
- Both serve the same purpose (agent browser control)
