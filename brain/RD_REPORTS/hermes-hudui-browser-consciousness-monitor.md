## What it is
Browser-based consciousness monitor for Hermes AI agent with persistent memory. Real-time WebSocket-updated dashboard with 10+ tabs for identity, memory, skills, sessions, health, costs, patterns, corrections, governance, and live chat.

## Key Features
- **10+ dashboard tabs:** identity, memory, skills, sessions, cron, projects, health, corrections, agents, profiles, patterns, timeline, governance, live chat
- **Per-model token cost tracking**
- **Multi-language:** English/Chinese
- **4 themes:** Neural Awakening, Blade Runner, fsociety, Anime (with CRT scanlines)
- **WebSocket real-time updates**
- **Reads Hermes data from ~/.hermes/** (same as TUI version)
- **Rich chat:** markdown, tool calls, reasoning display, streaming
