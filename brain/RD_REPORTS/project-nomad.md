## What It Is
Self-contained offline-first knowledge server. One curl command install on any Debian machine. Zero telemetry. Zero accounts. Zero cloud.

## What It Includes
| Layer | Tool | What You Get |
|------|------|-------------|
| AI Chat | Ollama + Qdrant | Local LLM + RAG with document upload |
| Knowledge Base | Kiwix | Offline Wikipedia, medical, survival guides |
| Education | Kolibri | Khan Academy courses, progress tracking |
| Maps | ProtoMaps | Offline OpenStreetMap regional maps |
| Data Tools | CyberChef | Encryption, encoding, analysis |
| Notes | FlatNotes | Local markdown note-taking |
| Management | Custom UI | Browser-based Command Center at port 8080 |
