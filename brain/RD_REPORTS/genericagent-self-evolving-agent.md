## What it is
Minimal (~3K lines core + 100-line agent loop), self-evolving autonomous agent framework for local machine control. Evolves skills from each task into a growing skill tree. Self-bootstrapping proof: setup and commits autonomously.

## Key Features
- **Self-evolving skill tree:** Task execution paths crystallize into reusable skills
- **9 atomic tools:** browser, terminal, filesystem, keyboard/mouse, screen vision, ADB mobile
- **Real browser injection:** preserves sessions, no headless mode needed
- **Cross-model:** Claude, Gemini, Kimi, MiniMax, others
- **Token efficient:** <30K context window with layered memory
- **Self-bootstrapping:** autonomous Git setup and commits
