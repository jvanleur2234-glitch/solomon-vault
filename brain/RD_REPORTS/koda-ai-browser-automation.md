## What It Does

AI-powered browser automation framework that interacts with web apps like a human. Blends multi-LLM support (Gemini, OpenAI, Claude) with computer vision and self-healing selectors.

Key capabilities:
- **Self-healing**: adapts to UI changes, avoids broken selectors
- **Multi-browser + mobile**: Chrome, Firefox, Safari, Edge; iOS/Android via WebDriverIO/Appium
- **Stagehand-compatible API**: drop-in replacement with familiar JS API
- **REST API hardening**: rate limiting, safer default host binding (127.0.0.1)
- **Tool hardening**: APITool blocks private/local targets by default; FileTool enforces path boundaries
