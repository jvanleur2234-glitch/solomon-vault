## What It Does
Self-evolving engine for AI agents that scans logs/errors to generate constrained prompt optimizations. Features four strategy modes (balanced growth, stability hardening, rapid feature addition, emergency fixes). Offline/EvoMap network support.

## Key Features
- Scans run logs and error patterns → extract key signals
- Matches evolution genes → generate constrained prompt optimizations
- Four modes: balanced growth, stability hardening, rapid feature addition, emergency fixes
- Offline mode available
- EvoMap network: collaborate with other nodes to share skills
- Safe: never modifies core business code
