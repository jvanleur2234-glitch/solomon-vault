## What It Does
Lightweight (~10MB binary), zero-setup browser automation toolkit for AI agents. Adheres to WebDriver BiDi standard (not proprietary protocols). Supports CLI, MCP server, and language libraries (JS/TS, Python, Java).

## Key Features
- ~10MB binary, auto-downloads Chrome
- WebDriver BiDi standard compliance (cross-browser compatible)
- CLI: `vibium go`, `vibium click @ref`, `vibium fill`, `vibium screenshot`, `vibium pdf`
- MCP server mode for agent integration
- JS/TS, Python, Java libraries
- Standards-based (transparent, not locked-in)

## For Solomon OS / ClawLess
- Direct competitor to browser-use, stagehand, HyperAgent
- Zero-setup appeal — single binary, auto-downloads browser
- WebDriver BiDi = more standards-compliant than proprietary CDP wrappers
- Lightweight alternative when full browser-use isn't needed
