## What It Does
Event-driven, multi-agent AI framework using Solace Event Mesh for scalable, decoupled, asynchronous workflows. Agents with distinct skills delegate tasks, share data, and execute multi-step workflows via an orchestrator.

## Key Features
- Event-driven architecture via Solace Event Mesh
- Agent-to-Agent (A2A) communication protocol
- Orchestrator coordinates task delegation
- Built on Google ADK and Solace AI Connector (SAC)
- Python-dominant with TypeScript/JS components
- REST APIs, Slack, web UIs, custom gateways
- 64 open issues, active development

## For Solomon OS / Hermes
- Alternative orchestration model (event-driven vs direct messaging)
- A2A protocol alignment with Hermes architecture
- Event mesh = better scalability for high-volume agent workloads
- Integration with Solace adds enterprise messaging backbone
