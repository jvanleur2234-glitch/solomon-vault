# nanobrowser — AI-Powered Chrome Extension Browser Automation (12,781 stars, Apache 2.0)

## Quick Summary
Open-source Chrome extension that lets AI agents automate real browser sessions with authenticated logins. Multi-agent system running entirely in-browser with your own API keys. Free alternative to OpenAI Operator.

## What It Does
- Runs autonomous, agent-driven workflows in Chrome with authenticated sessions
- Multi-agent collaboration for complex web tasks
- Privacy-first: local execution, no subscription fees
- Supports multiple LLM providers: OpenAI, Anthropic, Gemini, Ollama, Groq, Cerebras, Llama, OpenAI-compatible
- Interactive side panel for real-time status, task automation, conversation history
- Batch jobs, scheduled execution, execution history with diffs, visual dashboards
- Google Sheets integration via browser session (no OAuth)
- MCP server for AI client integration

## Key Features
- Exact target extraction (not screenshot-based)
- Reproducible runs
- Multi-scenario batch processing
- Scheduled monitoring
- CSV/JSON export
- Low-cost, text-based interaction
