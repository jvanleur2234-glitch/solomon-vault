# magnitude — Vision-First Browser Automation Framework

## What It Is
Apache-2.0 TypeScript browser automation using vision-grounded models (Claude Sonnet 4 preferred) to plan actions visually, not via DOM.

## Key Signals
- **License:** Apache-2.0
- **Lang:** TypeScript
- **Latest:** magnitude-test@0.3.13
- **Fork:** https://github.com/jvanleur2234-glitch/magnitude

## Core Features
- Vision-first: understands interfaces visually → plans pixel-precise actions
- Navigate, Interact (mouse/keyboard), Extract (structured data), Verify (visual assertions)
- 90% on Web Voyager benchmark
- Self-healing via vision (adapts to site changes)
- npx create-magnitude-app for bootstrapping
