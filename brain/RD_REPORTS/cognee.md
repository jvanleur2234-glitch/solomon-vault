## What It Does
Builds persistent memory for AI agents with just 6 lines of code. Combines vector search + graph databases. ECL pipeline (Extract, Cognify, Load) replacing traditional RAG. Supports 30+ data sources (docs, conversations, audio).

## Key Features
- 6-line API to build agent memory
- Vector search + graph database = connected, searchable, relationship-aware memory
- ECL pipeline: Extract → Cognify → Load
- 30+ data sources: documents, conversations, images, audio transcripts
- Ollama support for local models
- CLI + local UI
