## What It Is
End-to-end AI Agent Engineering Platform built on an open-source TypeScript framework. Build, deploy, operate intelligent agents with memory, tools, multi-step workflows, supervisor/sub-agent orchestration.

## Key Capabilities
- **Core Runtime (@voltagent/core):** typed agents with roles, tools, memory, model providers
- **Workflow Engine:** declarative multi-step automations
- **Supervisors & Sub-Agents:** coordinated task routing teams
- **Tool Registry & MCP:** typed tools with lifecycle hooks
- **Memory, RAG, Guardrails, Evals, Voice** — full production stack
- **VoltOps Console:** cloud/self-hosted observability
