## What It Does
Minimalist (~3,000 lines), self-evolving AI agent framework. 9 built-in tools for controlling computers (browser, terminal, files, mobile via ADB). Agents learn from tasks, solidify paths into reusable skills. Low context consumption, local evolution.

## Key Features
- Self-evolving: task execution paths → reusable skills (no manual coding)
- 9 built-in tools: browser, terminal, filesystem, mobile ADB
- Real browser retention of login state (inject browser)
- Layered memory: context consumption << standard frameworks
- Frontend bots: WeChat, DingTalk, custom
- Code commits even made by the agent itself
