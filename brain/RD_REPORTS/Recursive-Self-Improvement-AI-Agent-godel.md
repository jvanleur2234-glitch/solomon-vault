# ai-in-pm/Recursive-Self-Improvement-AI-Agent — Gödel-Inspired Self-Modifying Framework (MIT)

## Quick Summary
Python framework for recursive self-improving AI agents inspired by Gödel's incompleteness theorems. Enables agents to analyze and modify their own code at runtime for self-awareness and iterative optimization.

## What It Does
- **Dynamic runtime code modification**: agents modify own code
- **Recursive self-improvement** via policy optimization
- **Adaptive environment complexity**
- **Comprehensive logging/state tracking**
- **Multi-agent support**
- **Flexible objective functions**: environment-based or optimization-based
- **Multi-API integration**: OpenAI, Anthropic, Mistral, Groq, Google, Cohere

## Architecture
- GodelAgentManager: orchestrator
- godel_agent/agent.py: core logic
- environment.py: adaptive environment
- app.py: lifecycle management
