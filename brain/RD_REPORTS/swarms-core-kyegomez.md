## What It Is

Rust-based core execution engine for Swarms framework. Provides multi-threading, concurrency, async execution with zero-cost abstractions. Python bindings via PyPI (`swarms-core`). Designed for production-grade performance in swarm orchestration.

## Key Components

- Rust execution strategies for agent orchestration
- Python API via pydantic models
- Memory-safe, zero-GC performance for large fleet management
- Makefile-based dev workflow (build, test, lint, coverage)
