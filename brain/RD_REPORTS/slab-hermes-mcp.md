## What It Does
Elixir SDK implementing the Model Context Protocol (MCP) — client AND server implementations for tools-driven LLM interactions. Fork of cloudwalk/hermes-mcp with additional examples and community contributions.

## Key Features
- Full MCP client + server in Elixir
- Streamable HTTP transport
- Fault-tolerant concurrency via Elixir supervision patterns
- Plug/Phoenix integration for HTTP routing
- MIT licensed (unlike cloudwalk which uses Apache 2.0)
