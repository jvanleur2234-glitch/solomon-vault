## What It Is
OWASP Agentic Skills Top 10 (AST10) is the security standard for AI agent skills across major ecosystems (OpenClaw, Claude Code, Cursor/Codex, VS Code). It details the top 10 risks to agentic skills and provides evidence-based mitigations, prevention guidance, and cross-platform standards.

## Key Content
- 10 principal security risks for agentic skills
- Practical mitigation and response strategies
- MAESTRO mapping aligning risks with CSA's 7-layer threat model
- Universal skill format for cross-platform security design
- Case studies, threat intelligence, risk assessment guidance
- Incident response playbooks

## Key Risks Covered
1. Agent Authorization and Control Hijacking
2. Agent Critical Systems Interaction
3. Agent Goal and Instruction Manipulation
4. Agent Hallucination Exploitation
5. Agent Impact Chain and Blast Radius
6. Agent Memory and Context Manipulation
7. Agent Orchestration and Multi-Agent Exploitation
8. Agent Resource and Service Exhaustion
9. Agent Supply Chain and Dependency Attacks
10. Agent Knowledge Base Poisoning
