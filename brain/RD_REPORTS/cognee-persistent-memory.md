# RD Report: cognee — Persistent Memory for AI Agents

**Original:** `cognee-ai/cognee` | **License:** MIT | **Stars:** 15,900+ | **Lang:** Python

## What It Is
Builds a unified memory layer for AI agents combining vector search + graph databases. 6 lines of code to add persistent memory. Replaces traditional RAG with ECL (Extract → Cognify → Load) pipeline.

## Key Capabilities
- Vector search + graph database = unified memory layer
- ECL pipeline (Extract, Cognify, Load)
- 30+ data source connectors (conversations, files, images, audio, etc.)
- 15,900+ stars (validated high interest)
- CLI + local UI
- Ollama local model support
- Highly customizable
