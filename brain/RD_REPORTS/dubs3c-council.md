# Council — Multi-Agent Deliberation Framework

**Repo:** `dubs3c/council` | **License:** MIT | **Lang:** Python

## What It Is
Structured multi-agent debate framework. Four-phase process: Input → Proposal → Debate → Consensus.

## Key Capabilities
- Distinct personas: The Architect, The Critic, The Application Security Specialist
- Debate flow: revision, agreement, raising concerns, early consensus
- Moderator generates consensus document; final report saved with full transcript
- Per-agent LLM/provider config
- Structured, probability-weighted deliberation for explainable consensus
