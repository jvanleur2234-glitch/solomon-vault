**License:** MIT  

## What It Does
n8n community node integrating Claude Code autonomous agent capabilities into n8n workflows. Generates code from natural language in multiple languages, executes autonomous coding tasks with full Claude Code capabilities.

## Key Features
- Generate Code: Python, JavaScript, TypeScript, Java, C++, C#, Go, Rust, PHP, Ruby
- Execute Agent Task: full autonomy (read/write/edit files, execute commands, search)
- Flexible permission modes (readonly, local, sandbox, full)
- Tool selection: fine-tune which tools Claude can access
- Environment config: working directory, env vars, system prompts

## Comparison to n8n-nodes-mcp
- MCP bridges to MCP servers (including Hermes)
- This node bridges directly to Claude Code runtime
- Both are MIT licensed
