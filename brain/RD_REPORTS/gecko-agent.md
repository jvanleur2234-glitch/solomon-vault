## What It Does
AI-powered Chrome extension that provides an agentic, conversational browser automation assistant. Think → act → observe loop: analyze request, perform actions, observe results, repeat. Multi-model support via OpenRouter (Claude, GPT-4o, Gemini, Llama).

## Key Features
- Multi-step workflows via chat interface
- 9 built-in automation tools for common browser tasks
- Agentic loop (up to 40 iterations)
- Real-time streaming token-by-token responses
- Specialized automation: LinkedIn data extraction, Airtable interaction
- Activity log, conversation history, custom prompts

## For Solomon OS
- **Use for:** Browser automation for web scraping, data extraction, form filling
- **Alternative to:** HyperAgent, agent-browser, browserable, browserclaw-agent
- **Edge:** Chrome extension = easy deployment. Useful for consumer-facing browser tasks.
