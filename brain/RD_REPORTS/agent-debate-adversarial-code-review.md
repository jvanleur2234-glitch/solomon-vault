## What It Is
Multi-agent adversarial code review where 2-4 AI agents (Claude, Codex, Gemini, Copilot) collaboratively edit a single file in place. Uses strikethroughs, evidence citations (file:line), and a dispute log. Agents must converge or escalate.

## Key Features
- Realistic debate format with evidence-based claims + formal concessions
- Debates tracked in log, disputes resolved through convergence
- Plans + rounds control: manual vs auto debate, max rounds, optional plan phase
- Guardrails ensure all actions grounded in existing code
- Installable as script: `curl [remote install script]`
- Multiple agent options at install time
