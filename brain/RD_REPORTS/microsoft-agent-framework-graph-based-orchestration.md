# RD Report: Microsoft Agent Framework — Graph-Based Multi-Language Orchestration

## Summary
Microsoft's cross-language (Python + .NET/C#) graph-based agent orchestration framework. Data flows, streaming, checkpointing, human-in-the-loop, time-travel. DevUI for interactive debugging. AF Labs experimental (RL, benchmarking). 120+ contributors, 74 releases. python-1.1.0 (2026-04-21). MIT.

## What It Does
- **Graph-Based Orchestration**: Connect agents and deterministic functions with data flows
- **Cross-Language**: Python and .NET with consistent APIs
- **Data Flow Features**: Streaming, checkpointing, human-in-the-loop, time-travel
- **DevUI**: Interactive UI for development, testing, debugging
- **AF Labs**: Experimental benchmarking, RL, research initiatives
- **OpenTelemetry**: Distributed tracing for observability
- **Middleware**: Flexible request/response processing pipelines
- **Multi-Provider**: Various LLM providers, ongoing additions
