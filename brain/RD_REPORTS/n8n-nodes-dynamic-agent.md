# RD Report: n8n-nodes-dynamic-agent — Modular AI Agent Node for n8n

**License:** MIT  
**Stars:** active  
**Relevance:** n8n workflow automation, AI agent nodes  

## What It Is
A flexible, modular AI Agent node for n8n with mixed message configuration (system/user/assistant), dual config modes, token usage tracking, and optional sub-nodes for memory, tools, and parsing.

## Key Features
- Mixed message configuration in any order (drag-and-drop)
- Individual mode + JSON Array mode for bulk configuration
- Token usage and cost tracking per message type
- Max iterations, temperature, streaming, timeout controls
- AI sub-node integration: memory, tools, output parsers
- Fallback mode with mock responses
- MIT licensed

## Solomon OS Relevance
- **Workflow integration:** could integrate Hermes into n8n pipelines
- **Automation layer:** n8n is a popular automation tool — having Hermes skills as n8n nodes would expand reach
- **MIT licensed:** safe to fork and adapt

MIT licensed, useful for n8n integration. Fork for the Hermes n8n node library.
