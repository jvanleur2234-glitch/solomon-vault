## What It Does
Mozilla's AI-powered web automation tool in the Tabstack project. Natural language commands control browsers, navigate sites, fill forms, extract data. OpenAI API or OpenRouter integration.

## Key Features
- Natural language task execution ("Find latest Reuters news", "Get stock price")
- Node.js 22+, TypeScript, npm global install or npx
- Playwright-based browser drivers
- Safety guardrails to constrain behavior
- CLI + programmatic usage (WebAgent/PlaywrightBrowser)
- Config stored at `~/.config/pilo/config.json`
