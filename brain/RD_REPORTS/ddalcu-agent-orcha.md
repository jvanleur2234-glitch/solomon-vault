## What It Is
Declarative YAML-based multi-agent AI framework. Define agents, workflows, and knowledge stores in version-controlled YAML. Built-in Studio UI (dashboard, IDE, visual composer), P2P encrypted agent sharing (no API keys transmitted), MCP tool integration, SQLite+vec knowledge graph, Chromium browser sandbox (CDP), ReAct autonomous loops, Organizations with CEO agents + ticket lifecycle.

## Why It Matters for Solomon OS
- **Declarative = easy to teach clients** — YAML configs could be generated by Solomon OS for AI employee workflows
- **P2P sharing** — encrypted peer-to-peer LLM sharing without cloud = AgentFM competitor
- **Organizations + CEO agents** — autonomous ticket triage + scheduled heartbeat execution = matches Solomon Heartbeat pattern
- **Browser sandbox** — CDP Chromium + Vision Browser = direct competitor to Solomon Browser
- **Zero external deps** — SQLite+vec embedded vector store, no cloud dependencies
