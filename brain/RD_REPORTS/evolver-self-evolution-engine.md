## What it is
Genome Evolution Protocol (GEP) powered self-evolution engine for AI agents. Converts ad hoc prompt tweaks into auditable, reusable evolution assets (Genes, Capsules, Events). Core engine behind EvoMap network.

## Key Features
- **GEP Protocol:** Structured evolution with audit trail (genes.json, capsules.json, events.jsonl)
- **Auto-log analysis:** Scans runtime history, extracts signals, generates evolution prompts
- **Strategy presets:** balanced, innovate, harden, repair-only
- **Mutation + PersonalityState gating:** Protocol-constrained evolution
- **Human-in-the-loop:** Review mode for safe evolution
- **Platform hooks:** Cursor, Claude Code, OpenClaw integrations
