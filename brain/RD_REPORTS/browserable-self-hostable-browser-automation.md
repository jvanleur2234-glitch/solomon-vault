## What It Is
Open-source, self-hostable JavaScript/TypeScript library for building browser automation AI agents. Navigate sites, fill forms, click buttons, extract data. SDK for task definition + result waiting.

## Key Features
- AI-driven browser automation with remote browser provider integration (Hyperbrowser, Steel)
- Multi-LLM support (Gemini, OpenAI, Claude)
- Local dev deployment via Docker Compose
- UI dashboard at http://localhost:2001, docs at http://localhost:2002
- SDK: `npm install browserable-js`
- Task-based: define task, wait for results
