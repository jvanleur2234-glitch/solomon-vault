## What It Is
Fast, native Rust CLI tool for browser automation designed for AI agents. Self-contained, scriptable, no Playwright/Node.js daemon required. CSS + ARIA accessible selectors.

## Key Features
- Core commands: open, click, dblclick, focus, type, fill, press, keyboard, hover, select, check
- Traditional CSS selectors + accessible selectors (role-based)
- Real keystrokes (keyboard type), mouse interactions
- Snapshot for accessibility tree with refs
- Screenshot capture (page or full-page)
- Installation: npm global, Homebrew, Cargo, or source
- One-time Chrome download: `agent-browser install`
