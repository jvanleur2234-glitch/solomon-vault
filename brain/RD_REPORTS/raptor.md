## What It Is
Raptor is an autonomous general-purpose AI offensive/defensive security research framework powered by Claude Code. It enables agents to perform adversarial thinking, automated reconnaissance, exploit development, and defensive operations.

## Key Components
- **Offensive modules:** automated attack research, exploit development
- **Defensive modules:** security analysis, vulnerability assessment
- **Claude Code integration:** leverages Claude's reasoning for security research
- **MCP server compatible:** can be used as an MCP tool
