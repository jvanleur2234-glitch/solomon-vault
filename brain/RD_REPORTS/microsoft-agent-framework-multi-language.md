## What it does
Multi-language framework (Python + .NET/C#) for building, orchestrating, and deploying AI agents with graph-based orchestration, streaming, checkpointing, human-in-the-loop, and time-travel capabilities.

## Key features
- Python + C# with consistent APIs
- Graph-based workflows: connect agents + deterministic functions via data flows
- Streaming, checkpointing, human-in-the-loop, time-travel debugging
- DevUI: interactive development/testing/debugging UI
- AF Labs: experimental packages (benchmarking, RL, research)
- Middleware system for request/response pipelines
- OpenTelemetry integration
- Multiple LLM providers
- Migration guides from Semantic Kernel + AutoGen

## Why it matters
Microsoft backing = enterprise-grade. Graph-based orchestration with time-travel debugging is unique. Strong migration path from Semantic Kernel/AutoGen = easy enterprise adoption.
