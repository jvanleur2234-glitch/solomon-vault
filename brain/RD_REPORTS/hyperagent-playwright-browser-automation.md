# hyperbrowserai/HyperAgent — AI-Powered Playwright Browser Automation

## Quick Summary
AI-powered extension for Playwright that enables natural-language browser automation. Blends LLM-driven commands with traditional Playwright actions for scalable, AI-assisted browsing and scraping.

## What It Does
- **page.ai()**: higher-level AI actions (deprecated in favor of page.perform)
- **page.extract()**: extract structured data per defined schema
- **executeTask()**: perform end-to-end tasks described in natural language
- **Action caching**: record-and-replay workflows deterministically
- **Stealth mode**: built-in anti-bot patches to reduce detection
- **Cloud-ready**: scale across many sessions via Hyperbrowser
- **MCP client**: integrates with tools (e.g., Composio)
