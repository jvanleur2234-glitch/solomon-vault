## What It Is
OWASP-grounded collection of step-by-step plays for security engineering tasks that AI agents can execute. Not a code library — procedures, checklists, decision criteria, and output templates for security assessments.

## Key Capabilities
- **15 security skills** covering code and infrastructure security
- **CWE, ASVS, WSTG, NIST 800-53 cross-standard traceability** via OpenCRE
- **Claude Code plugin integration** (recommended) + standalone plays
- **Structured outputs:** severity, CWE mappings, evidence, remediation, code examples
- **Skills catalog:** code-review-security, sca-audit, etc.

## Why It Matters for Solomon OS
Maps to AgentArmor Studio's 8-layer security framework. The OWASP plays define what a well-governed agent security practice looks like. Cross-reference with OWASP ASI01–ASI10. Could inform Hermes security skill documentation.
