# dapr/dapr-agents — Durable AI Agent Workflows (Apache 2.0, 662 stars)

## Quick Summary
Python-based framework for building scalable, autonomous AI agent systems with built-in workflow orchestration, security, statefulness, and telemetry. Extends Dapr for reasoning, acting, and collaboration among LLM-based agents.

## What It Does
- **Durable execution workflow engine**: guarantees task completion amid failures
- **Transparent distribution** across fleets
- **Multi-agent coordination**
- **Kubernetes-native deployment**
- **Data-source integration**: databases, documents, unstructured data
- **Secure and observable agent collaboration**
- **Vendor-neutral, open-source**

## Architecture
- Durable-execution workflow engine
- Fault tolerance and workflow guarantees
- Platform-ready resources with access controls
