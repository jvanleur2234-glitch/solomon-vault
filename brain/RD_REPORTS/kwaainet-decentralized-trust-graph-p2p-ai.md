# RD Report: KwaaiNet — Decentralized Trust Graph + P2P AI Infrastructure

## Summary
KwaaiNet by Kwaai-AI-Lab is a decentralized AI infrastructure project for user ownership of compute, storage, data. W3C verifiable credentials, did:peer identities, Ed25519 signatures, time-decayed trust scoring. DHT propagation. Rust CLI. Pre-built binaries (macOS, Linux, Windows). MIT.

## What It Does
- **Decentralized Trust Graph (DTG)**: Trust built via W3C verifiable credentials
- **did:peer Identities**: Decentralized identity management
- **Ed25519 Signatures**: Cryptographic trust attestation
- **Time-Decayed Trust Scoring**: Trust degrades over time
- **P2P Discovery**: DHT with bootstrap peer failover
- **Rust CLI**: Native kwaainet binary
- **Cross-Platform**: macOS, Linux, Windows binaries

## Tech Stack
- Language: Rust (primary), TypeScript, Shell
- License: MIT
- Latest: v0.3.24 (2026-03-20)

## Strategic Fit for Solomon OS

**WATCH** — Trust graph for distributed AI compute selection. W3C credentials align with enterprise compliance.

Key learnings:
1. **Time-Decayed Trust**: Prevents stale trust judgments
2. **W3C Verifiable Credentials**: Industry standard for trust
3. **Bootstrap Failover**: Resilience for P2P discovery
4. **did:peer**: Decentralized identity without central authority
