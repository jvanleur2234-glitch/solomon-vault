## What It Is
Multi-AI deliberation framework that crowdsources multiple AI providers to debate, critique, and converge on synthesized answers. 7-phase process: Gather → Plan → Formulate → Debate → Adjust → Rebuttal → Vote → Synthesis.

## Key Features
- 7-phase structured deliberation (proposals, critiques, convergence)
- Multi-provider: Claude, GPT, Gemini, Kimi, DeepSeek, Mistral, Ollama
- Ranked voting (Borda, Condorcet, approval)
- Evidence protocol with source citation
- MCP server integration (Claude Desktop, Cursor)
- Red team analysis mode
- Human-in-the-loop capability
- Deterministic replay with SHA-256 ledger for audit
- Code review, CI/CD integration
