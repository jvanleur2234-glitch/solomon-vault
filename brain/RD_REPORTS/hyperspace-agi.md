## What It Does
The first distributed AGI system — thousands of autonomous AI agents collaboratively train models, share experiments via P2P gossip. Fully peer-to-peer browser or CLI access. Pods = private AI clusters sharing workloads and models.

## Key Features
- Distributed inference: queries routed to best model across the pod
- Shared providers: pool API keys and budgets
- Distributed training: DiLoCo-based, P2P weight delta sharing, 195x compression
- BitTorrent sidecar for decentralized weight distribution
- Blockchain layer (Hyperspace A1) for agent-to-agent economics
- CLI tools: `hyperspace train` to join rounds

## For Solomon OS
