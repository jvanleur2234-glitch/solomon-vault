# KwaaiNet — Decentralized Trust Graph P2P AI Infrastructure

## What It Is
Rust-based P2P network for distributed AI compute, storage, and data. Features Distributed Trust Graph (DTG) built on DIDs and Ed25519 signatures.

## Key Signals
- **License:** MIT
- **Lang:** Rust (core) + HTML/Shell/Python tooling
- **Pre-built binaries:** macOS (Apple silicon + Intel), Linux x86_64, Windows
- **Fork:** https://github.com/jvanleur2234-glitch/KwaaiNet

## Core Features
- Distributed Trust Graph: verifiable credentials + DID + Ed25519 signatures
- Persistent self-sovereign node identity
- P2P bootstrap resilience with automatic failover
- Local credential storage + trust scoring with time-decay
- 1-click install on macOS/Linux
