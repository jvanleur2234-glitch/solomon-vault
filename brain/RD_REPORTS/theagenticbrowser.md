## What It Does

Python-based AI agent framework for automating browser tasks. Orchestrates three-agent loop: Planner → Browser Agent → Critique Agent for iterative refinement.

**Key Components:**
- Planner Agent: breaks down tasks into actionable steps
- Browser Agent: performs real browser actions
- Critique Agent: evaluates results, verifies success
- Cross-domain data correlation and workflow feedback loops
