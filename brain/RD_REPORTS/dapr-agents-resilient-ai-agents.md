# RD Report: Dapr Agents — Durable Workflow AI Agent Framework

## Summary
Python framework for scalable, autonomous AI agents with built-in workflow orchestration, observability, and stateful execution. Built on Dapr for Kubernetes-native resilience.

## What It Is
Durable-execution workflow engine. Runs thousands of agents efficiently with transparent distribution. Automatic retries, fault tolerance, state recovery. Built-in telemetry and security.

## Key Features
- **Durable execution**: workflows persist and recover automatically
- **Scale**: thousands of agents with distributed execution
- **Resilience**: automatic retries, guaranteed task completion
- **Observability**: built-in telemetry
- **Kubernetes-native**: easy K8s deployment
- **Multi-agent collaboration**: secure by default
- **Data integration**: databases, documents, diverse sources
- **Apache 2.0**
