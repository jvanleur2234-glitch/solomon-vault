## What It Is

Wingman is an autonomous AI agent from **Emergent Labs** (founder @mukundjha) that runs **inside messaging apps** — WhatsApp, Telegram, iMessage, Slack — and handles real work in the background. It's not a chatbot you prompt; you set it once, it runs on autopilot.

**Core features:**
- **Multi-agent network** — deploy multiple Wingmen for different roles (email triage, meeting prep, repo research, follow-ups)
- **Daily 8 AM briefing** via your preferred messaging app
- **Tool integrations:** GitHub, Slack, Notion, email, CRMs
- **Actions:** repo reviews, code commits, email drafting, job finding, subscription management, CRM updates
- **Context persistence** — carries memory over time, no restart-from-scratch

**Demo highlights:**
- Reviewed a GitHub repo, suggested fixes, committed code — all from inside WhatsApp
- Three Wingmen working in parallel: email triage, meeting prep, follow-up triggers
- Only interrupts before sensitive actions
