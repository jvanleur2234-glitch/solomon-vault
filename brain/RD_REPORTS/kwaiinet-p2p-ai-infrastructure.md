## What it does
KwaaiNet is a Rust-based, decentralized AI infrastructure aiming to run AI compute and data storage in a peer-to-peer (P2P) network.

Key features:
- Decentralized AI compute and data ownership: Users control their own compute, storage, and data
- P2P networking: Uses libp2p and a P2P stack for connections
- Sovereign identity and trust: Implements a Decentralized Trust Graph (DTG) with W3C Verifiable Credentials, did:peer:DIDs, Ed25519 signatures
- Persistent node identity: Each node uses a permanent Ed25519 keypair
- Pre-built binaries for macOS, Linux, and Windows
- MIT licensed
