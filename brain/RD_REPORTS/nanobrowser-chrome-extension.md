## What It Does

Chrome extension + MCP server for AI-powered in-browser web automation. Multi-agent system for autonomous workflows with local execution.

**Key Features:**
- Multi-agent collaboration for web workflows
- Local browser execution (no cloud data leakage)
- Multiple LLM providers (OpenAI, Anthropic, Gemini, Ollama, Groq, Cerebras, Llama)
- Interactive UI with real-time status
- Cross-website task automation
- Open-source alternative to OpenAI Operator
