## What It Is
Enterprise-grade autonomous agent hedge fund using swarm intelligence. Combines specialized AI agents (Director, Quant, Risk, Execution) for end-to-end market analysis, risk management, and execution on Solana.

## Key Features
- **Multi-Agent Architecture**: 4 specialized agents for trading pipeline
- **Real-time market analysis**: Live data integration
- **Risk-first design**: Built-in position sizing and risk assessment before execution
- **Structured JSON outputs**: For downstream systems
- **Extensible**: Modular design for custom strategies and venues
- **Solana-native**: Currently supports Solana, Coinbase coming
