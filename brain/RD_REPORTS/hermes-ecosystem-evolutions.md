- **Forked:** jvanleur2234-glitch/hermes-agent-camel (pending push due to network)

## hermes-skill-factory — Meta-Skill Generator
- **Cloned:** /home/workspace/hermes-skill-factory
- **What:** Agent runs task → automatically generates new Hermes skill. "Agent builds its own weapons."
- **Why it matters:** Self-improving loop at the skill level. Hermes learns new capabilities without human intervention.
- **LINK fit:** ★★★★★ — #self-improvement #hermes #automation

## maestro — Long-Running Agent Framework
- **What:** Structured memory + plan-approve-execute for persistent agent tasks. Hermes evolves into a long-duration agent.
- **Why it matters:** Enables agents that run for hours/days, not just single turns.
- **LINK fit:** ★★★★☆ — #long-running #memory #hermes

## icarus-plugin — Self-Training Succession
- **What:** Agent works + trains its replacement ("successor agent"). "Agent retires and trains its replacement."
- **Why it matters:** The most advanced self-improvement pattern. Eliminates the need for human retraining.
- **LINK fit:** ★★★★★ — #self-improvement #succession #hermes
