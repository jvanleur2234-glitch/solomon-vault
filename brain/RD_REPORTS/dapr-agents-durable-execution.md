## What It Does
Python-based framework for scalable, autonomous AI agent systems built on Dapr. Provides durable-execution workflow orchestration, Kubernetes-native deployment, fleet-wide agent lifecycle management, built-in telemetry, and multi-agent collaboration. Statefulness by default — tasks survive interruptions.

## Key Components
- **Durable execution engine** — retries, state recovery across interruptions
- **Kubernetes-native** — fleet management, auto-scaling agent deployments
- **Data-driven** — easy integration with databases, docs, unstructured data
- **Telemetry by default** — distributed tracing, observability
- **Python primary** — active v1.0.1 release
