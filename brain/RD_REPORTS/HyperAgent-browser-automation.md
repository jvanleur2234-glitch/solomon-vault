# HyperAgent — Natural Language Browser Automation (Apr 25, 2026)

**Fork:** `jvanleur2234-glitch/HyperAgent` (MIT)
**Source:** https://github.com/hyperbrowserai/hyperagent

## What It Does
Playwright-based AI browser automation with:
- `page.ai()` for natural language commands
- `page.extract()` for structured data via Zod schemas
- `page.perform()` for rapid granular actions
- Stealth mode to bypass bot detection
- Action caching for deterministic replay
- Cloud-ready via Hyperbrowser
- MCP client for Google Sheets etc.
