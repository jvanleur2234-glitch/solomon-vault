## What It Does
Open-source multi-agent deliberation platform: runs several AI coding agents in parallel, peer-reviews outputs anonymously, synthesizes superior answer. Parallel execution → blind peer review → synthesis.

## Key Features
- Multi-Agent Execution: run several agents in parallel (Claude Code, Codex, OpenCode)
- Anonymous Peer Review: juror models critique without bias
- Consensus-Based Ranking: objective scoring to filter hallucinations
- CLI + Electron desktop GUI
- Local-first: data stays on machine
- ~3 minutes per prompt for full workflow
