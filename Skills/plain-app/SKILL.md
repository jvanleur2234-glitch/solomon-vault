# PlainApp — Android Phone Hub as Solomon Air Mobile Layer

Source: github.com/plainhub/plain-app (10.3K stars, Apache 2.0)

## What It Does
- Full Android phone management from any browser
- Calendar, contacts, notes, SMS, calls, photos
- No cloud — all data stays on device
- Android app + web dashboard

## For Solomon Air
PlainApp = the mobile brain layer:
- Solomon Air VoIP integrates into PlainApp
- User's phone becomes an AI agent endpoint
- JackConnect works from phone SMS/calls
